# Flask To-Do List 📝
A web-based task management app built with Python and Flask.

## Description
A simple To-Do List application where you can manage your daily tasks from the browser. Tasks can be added, edited, and deleted through a clean web interface powered by Flask.

## Features
* Add new tasks
* Edit existing tasks
* Delete tasks

## How to Run

```bash
pip install flask
python app.py
```

Then open your browser and go to `http://127.0.0.1:5000`
